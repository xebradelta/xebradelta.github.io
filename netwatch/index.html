    Scope: a web page can only observe what the browser exposes — it monitors this device's connection security, not your whole LAN. It cannot capture packets, scan other machines, or see router traffic. For network-wide monitoring pair this with a dedicated tool on the router itself (e.g. ntopng, Pi-hole, or Suricata). Probes go to Cloudflare (1.1.1.1), Google (gstatic.com), and GitHub's CDN (githubassets.com); IP lookup uses ipify.org and ipapi.co. Nothing is stored or sent anywhere else.
